# DUDUGAME

### 과정
처음에는 2D 두더지 게임을 구성하였습니다.

그러나 2D에서 3D로 업그레이드 하기 위하여 [opencv](https://github.com/opencv/opencv)에서 카메라를 통해 손을 인식할 수 있는 프로그램을 얻었습니다. 

이후 카메라를 이용하여 화면에 나오는 두더지들을 직접 손으로 잡게 하는 두더지 게임을 완성시켰습니다.
